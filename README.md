# kaltura-player-rn

React Native integration of Kaltura player

## Installation

```sh
npm install kaltura-player-rn
```

## Usage

```js
import KalturaPlayerRn from "kaltura-player-rn";

// ...

const result = await KalturaPlayerRn.multiply(3, 7);
```
